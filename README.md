# Just Travel

This project is a front-end challenge that objective is build a pagination ticket listage.

## Built with

This project was built with:

- NextJs
- Stitches
- Ant Design
- Typescript
- Redux
- Axios

## How run project

To development:

```bash
yarn dev
```

To deploy:

```bash
yarn build
yarn start
```

Now we can access the application in this url: http://localhost:3000
